```c
#include <stdio.h>

int main() {
  	char username[] = "jameslyne";
  	printf("Your username is %s \n", username);
    return(0);
}
```

| Command | Effect         |
| ------- | -------------- |
| %s      | insert string  |
| %d      | insert int     |
| %ld     | long           |
| %hd     | short          |
| %lu     | unsigned       |
| %hu     | unsigned short |
| %u      | unsigned int   |
| %c      | character      |
| %x      | hexadecimal               |
